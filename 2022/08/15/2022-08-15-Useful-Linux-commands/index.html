[Note] useful Linux commands

Author: Slay Date: August 15, 2022  0:29:00 Category: Note

find

  • 파일 이름의 전체나 파일 이름의 일부로 검색

  • 파일 경로에서 매칭되는 것을 검색

  • 파일 유형으로 검색

  • 파일 크기 기준으로 검색

  • 파일 생성(수정) 시간 기준으로 검색

  • 특정 파일보다 더 나중에 생성(또는 수정)된 파일을 검색
    ex) 어떠한 작업을 한 후 이로 인해 변경된 파일이 무엇이 있는지 찾아볼 때 이용

  • 특정 파일보다 더 이전에 생성(또는 수정)된 파일을 검색

  • 위의 옵션들을 and, or 연산자로 묶어서 검색

tail

  • 텍스트 파일의 끝 부분 내용을 확인할 때

  • 표시할 라인 수를 지정

  • 파일 끝 부분 내용을 출력한 후 이후 추가된 파일 내용을 계속 출력

grep

  • 파일에서 어떠한 키워드를 검색

  • 몇 번 매칭이 되었는지만 횟수를 셀 때

  • 검색된 키워드의 라인 수 표시

  • 어느 파일에서 찾았는지 알고 싶을 때 파일명 표시

  • 현재 및 하위 디렉토리 내 모든 파일에서 검색

  • 검색한 키워드의 위, 아래 n번 줄에 해당하는 내용도 출력

  • 특정 키워드를 제외할 때

  • 특정 라인을 제거할 때

sed

  • 특정 파일에서 특정 단어를 바꾸거나, 어떤 라인만 삭제할 때

  • 수정하고자 하는 파일을 바로 수정
